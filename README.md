# Installation
```bash
$ nimble install climinesweeper
```

# Usage
```bash
$ climinesweeper
```
```
option:
  -v, --version        display the version
  -h, --help           display the help
  <5 <= number <= 20>  Set the number of cells and start the game
```

# Requirement
- nim >= 1.6.8
- illwill >= 0.3.0
